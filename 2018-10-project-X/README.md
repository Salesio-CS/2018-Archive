## How to use it
> Attention: We'll need to start backend first.

### backend
1. move to `backend` folder
1. `.envtemplate` rename to `.env` and fill properties
1. `npm install`
1. `npm run build`
1. `npm start`

### frontend
1. move to `frontend` folder
1. `npm install`
1. `npm run build`
1. `npm start`
